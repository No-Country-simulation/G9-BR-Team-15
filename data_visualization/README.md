\# ⚡ Lu | Men Dashboard



\## 📌 Objetivo



Dashboard desenvolvido para analisar padrões de consumo energético,

identificar oportunidades de eficiência e apoiar decisões baseadas em dados.



\---



\## 🤖 Inteligência Artificial



O projeto utiliza classificação energética para categorizar clientes em:



🟢 Eficiente  

🟡 Moderado  

🔴 Ineficiente



\---



\## 📊 Indicadores analisados



\- Total de clientes analisados

\- Consumo total de energia (kWh)

\- Consumo médio por cliente

\- Quantidade média de equipamentos

\- Tempo médio diário de utilização

\- Percentual de clientes eficientes
